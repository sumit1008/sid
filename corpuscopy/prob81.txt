J. Xorderable Array
You are given an array A of N integers: [A1,A2,…,AN].
The array A is (p,q)-xorderable if it is possible to rearrange A such that for each pair (i,j) that satisfies 1≤i<j≤N, the following conditions must be satisfied after the rearrangement: Ai⊕p≤Aj⊕q and Ai⊕q≤Aj⊕p. The operator ⊕ represents the bitwise xor.
You are given another array X of length M: [X1,X2,…,XM]. Calculate the number of pairs (u,v) where array A is (Xu,Xv)-xorderable for 1≤u<v≤M.
